## 🔥 '개인사'의 프로젝트 중점사항

- 대용량 트래픽에 유연하게 대응 할 수 있는 서버를 만들자
- 깔끔하고 재활용 가능한 코드로 이루어진 클라이언트를 만들자
- 안전한 서비스를 위한 테스트 코드를 작성하자
